HTML5 e CSS:

HTML5:

HTML 5 é a quinta e atual versão do HTML. Ele melhorou a marcação disponível para documentos e introduziu interfaces de programação de aplicativos (API) e Document Object Model (DOM).

Elementos HTML5
Elementos Usados Descrição
Main done Indica conteúdo principal e único, que não se repete.
Section done Especifica uma parte do conteúdo, usada em conjunto com etiquetas h1|h2|…
Nav done Indica menu de navegação com links para as páginas
Header done Indica cabeçalho
Footer done Indica rodapé da secção
Article clear Indica porção do documento que pode ser vista de forma independente.
Aside clear Indica anotações, publicidade ou outros assuntos desconexos.

CSS:

CSS é a linguagem que usamos para definir o estilo de um documento HTML. CSS descreve como os elementos HTML devem ser exibidos. Este tutorial ensinará CSS do básico ao avançado..

Tipos de seletores CSS
Elementos Usados Descrição
Universal done Aplica o estilo a todos os elementos da pagina
Tipo done Aplica o estilo do nome do marcador a utilizar
Classe clear Aplica o estilo do nome de uma classe a utilizar
Identificador done Aplica o estilo consoante o Identificador(muito util em Javascript)
Atributos done Aplica o estilo consoante o atributo de um Elemento
Pseudo-classe clear Aplica o estilo consoante o estado de um elemento
Pseudo-elemento clear Aplica o estilo consoante em partes de elementos


Combinadores de seletores CSS
Elementos Usados Descrição
Seletores descendentes done div p– todos os "p" que estão "dentro" de um div
Seletores filhos done div > p– todos os "p" que estão "logo dentro" de um div
Seletores adjacentes done div ~ p– todos os "p" que estão colocados depois de um "div"
Seletores imediatamente adjacentes done div + p– todos os "p" que estão colocados imediatamente depois de um "div"
Agrupação done div,p– os div e p ficaram com o mesmo estilo
